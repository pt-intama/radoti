# RadotiJS

RadotiJS is an application built on NestJs for storing files as a central file storage on microservices architecture.

## Getting started

```bash
$ git clone https://github.com/pt-intama/radoti.git
$ cd radoti
$ yarn setup
```

## Running the app

```bash
$ yarn start
```
